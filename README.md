# RGB editor fotek

Tento program umožňuje uživatelům kombinovat RGB barvy pro úpravu fotografií.

## Popis

Program umožňuje uživatelům vybrat jeden ze dvou obrázků - pralesničku batikovou (D) nebo vosu (V) - a následně upravit barvy obrázku.

## Použití

1. Zvol `D` nebo `V` podle obrázku, který chceš upravovat.
2. Vyber barvu, kterou chceš upravit (červená, zelená, modrá nebo všechny barvy).
3. Zadej požadované hodnoty pro úpravu barev.
4. Program vygeneruje upravený obrázek a zobrazí ho, (Měl by se i uložit).
